Simple mapping app

This is an example of a simple mapping app created with the GIS Cloud JS API.

The code: 

giscloud.ready(function() {

    // create a viewer
    var viewer = new giscloud.Viewer("mapViewer", 1);

    // create a toolbar
    var toolbar = new giscloud.ui.Toolbar({
       viewer: viewer,
       container: "toolbar",
       defaultTools: ["pan", "zoom", "full", "measure"]
    });

});
